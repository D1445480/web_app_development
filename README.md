## 系統簡介

本系統旨在解決記帳不便與資訊不透明的問題，透過簡潔直觀的操作介面與清楚的數據呈現，幫助使用者輕鬆建立記帳習慣，並有效管理個人財務。

---

## 技術棧

- **前端：** Jinja2
- **後端：** Python + Flask
- **資料庫：** SQLite
